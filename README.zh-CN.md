# 前端设计与开发课程指南

本仓库包含**前端设计与开发**课程的学习文档，专为大专院校一年级新生量身定制。
内容基于教材《网页设计与制作项目教程（HTML5+CSS3+JavaScript）（AI助学）（第3版）》。

[阅读英文版 (English)](./README.md)

## 特点
- **项目驱动学习**：8个实际项目全面覆盖HTML5、CSS3和JavaScript核心知识点。
- **AI辅助学习**：引入CodeGeeX等AIGC工具，帮助学生掌握前沿AI编程技巧，提升开发效率。
- **双语支持**：基于VitePress国际化功能，提供中英文双语文档。

## 本地运行

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run docs:dev

# 构建生产版本
npm run docs:build
```
