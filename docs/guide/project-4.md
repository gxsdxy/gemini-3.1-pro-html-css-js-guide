# 项目4 “爱家居”企业网站首页制作

网页排版不仅仅是文字和颜色，更是对网页元素的排列组合。本项目将带你深入理解网页布局的核心机制——**盒子模型 (Box Model)**，并学习浮动与定位技术，最终完成一个结构复杂的企业网站首页。

---

## 任务 4-1 认识盒子模型

在 CSS 中，所有的 HTML 元素都可以看作是一个矩形的盒子。理解盒子模型，是学好网页布局的关键。

一个完整的盒子模型由内到外由四个部分组成：
1. **内容 (Content)**：盒子里面装的东西（文字、图片等）。
2. **内边距 (Padding)**：内容与边框之间的距离（相当于快递盒里的缓冲泡沫）。
3. **边框 (Border)**：盒子的边缘（相当于快递盒的纸板）。
4. **外边距 (Margin)**：盒子与周围其他盒子之间的距离。

---

## 任务 4-2 盒子模型基础属性

让我们来详细了解如何设置盒子的各个部分：

### 1. 边框 (border)
* `border-width`：边框粗细。
* `border-style`：边框样式（`solid` 实线, `dashed` 虚线, `dotted` 点线）。
* `border-color`：边框颜色。
* **复合写法**：`border: 1px solid #ccc;`

### 2. 内边距 (padding)
* `padding: 10px;`（上下左右均为10px）
* `padding: 10px 20px;`（上下10px，左右20px）
* `padding: 10px 20px 30px 40px;`（上、右、下、左，顺时针方向）

### 3. 外边距 (margin)
用法与 padding 类似。
> [!TIP] 实用技巧
> 使用 `margin: 0 auto;` 可以让一个有固定宽度的块级盒子在其父元素中**水平居中**。

### 4. 宽度和高度 (width, height)
* 这两个属性设置的是**内容 (Content)** 的尺寸，而不是整个盒子的尺寸。
* 盒子实际总宽度 = width + 左/右 padding + 左/右 border + 左/右 margin。

### 5. 背景属性 (background)
* `background-color`：背景颜色。
* `background-image`：背景图片。
* `background-repeat`：背景是否平铺（`no-repeat` 不平铺）。
* `background-position`：背景图片位置。

## 任务 4-3 盒子模型新增属性 (CSS3)

CSS3 为盒子模型带来了一些非常漂亮的视觉特效。

### 1. 圆角 (border-radius)
让盒子的直角变成圆角。
* `border-radius: 10px;` 
* 如果是一个正方形盒子，设置 `border-radius: 50%;` 会把它变成一个圆形。

### 2. 阴影 (box-shadow)
`box-shadow: 5px 5px 10px rgba(0,0,0,0.3);` (水平偏移，垂直偏移，模糊半径，颜色)。

### 3. 渐变 (gradient)
可以设置背景颜色的线性渐变 (`linear-gradient`) 或径向渐变。

### 4. 透明度 (opacity)
设置元素的不透明度，范围从 `0` (完全透明) 到 `1` (完全不透明)。

## 任务 4-4 元素的类型与转换

HTML 元素主要分为两类：
* **块级元素 (Block)**：如 `<div>`, `<p>`, `<h1>`。独占一行，可以设置宽高。
* **行内元素 (Inline)**：如 `<span>`, `<a>`。不会独占一行，设置宽高无效。
* **行内块元素 (Inline-block)**：如 `<img>`。不独占一行，但可以设置宽高。

**元素转换**：使用 `display` 属性。
* `display: block;` (转为块级)
* `display: inline-block;` (转为行内块)

---

## 任务 4-5 元素的浮动 (Float)

要让多个块级元素并排显示（比如导航栏的几个菜单项），最经典的方法就是使用浮动。
* `float: left;`（向左浮动）
* `float: right;`（向右浮动）

**清除浮动**：浮动的元素会脱离标准文档流，导致父元素高度塌陷。常用的清除浮动方法有：
1. 额外标签法：`<div style="clear: both;"></div>`。
2. 父元素设置 `overflow: hidden;`。
3. **推荐做法**：使用 `:after` 伪元素清除浮动。

---

## 任务 4-6 元素的定位 (Position)

当我们需要把一个元素固定在屏幕的某个角落，或者让元素重叠显示时，就需要使用定位。
* **static (静态定位)**：默认值，没有定位。
* **relative (相对定位)**：相对于自己原来的位置进行移动，原位置保留。
* **absolute (绝对定位)**：相对于最近的**已定位**祖先元素进行移动，脱离文档流。
* **fixed (固定定位)**：相对于浏览器窗口进行定位，页面滚动时它不动。

> [!IMPORTANT] 核心口诀
> **子绝父相**：子元素使用绝对定位，父元素使用相对定位。这是网页开发中最常用的定位组合！

## 任务 4-7 至 4-11 实战：“爱家居”首页制作

结合浮动和定位，我们将完成“爱家居”企业网站首页：
1. **导航和 Banner**：使用浮动制作水平导航，相对定位包裹 Banner 图。
2. **热门推荐**：使用浮动将多个商品卡片并排排列，结合盒子阴影增加立体感。
3. **悬浮框**：使用固定定位 (`fixed`) 制作一个始终在屏幕右侧的“在线客服”悬浮按钮。
