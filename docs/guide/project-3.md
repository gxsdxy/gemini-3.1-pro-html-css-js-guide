## 任务 3-1 CSS 核心基础

CSS 的出现，使得网页的内容与样式得以分离。

### 1. 结构与表现分离
将 HTML (结构) 和 CSS (表现) 分开的好处是：
* **代码更清晰**：HTML 文件里只有标签，没有繁杂的样式代码。
* **维护更方便**：改一个 CSS 文件，就能改变整个网站的风格，而不用逐个修改 HTML 页面。

### 2. CSS 样式规则
CSS 规则由两个主要部分构成：**选择器 (Selector)** 和 **声明块 (Declaration Block)**。
```css
h1 {
    color: red;
    font-size: 24px;
}
```
* `h1` 是选择器，表示要选中页面上所有的 `<h1>` 标签。
* `{ ... }` 是声明块，里面包含一条条具体的样式属性和值（用冒号分隔，分号结尾）。

### 3. CSS 样式的引入方式
有三种方式可以将 CSS 引入到 HTML 中：
* **行内样式**：直接写在标签的 `style` 属性里（不推荐，违背分离原则）。
  `<p style="color: blue;">蓝色文字</p>`
* **内部样式表**：写在 HTML `<head>` 里的 `<style>` 标签中。
* **外部样式表（强烈推荐）**：写在一个独立的 `.css` 文件中，然后在 HTML 中通过 `<link>` 标签引入。
  `<link rel="stylesheet" href="style.css">`

### 4. CSS 基础选择器
最常用的三大基础选择器：
* **标签选择器**：选中所有同名标签，例如 `p { ... }` 选中所有段落。
* **类选择器 (Class)**：以 `.` 开头，选中所有带有指定 `class` 属性的元素。
  * HTML: `<div class="box"></div>`
  * CSS: `.box { ... }`
* **ID 选择器**：以 `#` 开头，选中拥有特定 `id` 属性的唯一元素（一个页面中 ID 不能重复）。
  * HTML: `<div id="header"></div>`
  * CSS: `#header { ... }`

## 任务 3-2 CSS 文本样式属性

就像排版 word 文档一样，CSS 可以精细地控制文字的显示。

### 1. CSS 字体样式属性
* `font-family`：设置字体系列（如 "Microsoft YaHei", Arial）。
* `font-size`：设置字体大小（常用单位：px）。
* `font-weight`：设置字体粗细（如 bold, 400, 700）。
* `font-style`：设置字体倾斜（italic）。

### 2. CSS 文本外观属性
* `color`：文本颜色。
* `text-align`：文本对齐方式（left, center, right）。
* `text-decoration`：文本装饰线（underline 下划线, none 去除超链接默认下划线）。
* `line-height`：行高，常用来实现单行文字的垂直居中。
* `text-indent`：首行缩进（常用单位：em，`2em` 表示缩进两个中文字符）。

## 任务 3-3 CSS 高级特性

随着页面越来越复杂，你需要更灵活地选择元素。

### 1. CSS 复合选择器
* **后代选择器 (空格)**：`.nav li { ... }` 选中类名为 nav 里面的所有 li 标签。
* **子代选择器 (>)**：`.nav > li { ... }` 只选中直接子元素 li，不包括孙子元素。
* **并集选择器 (逗号)**：`h1, h2, h3 { ... }` 选中 h1, h2, h3，为它们应用相同的样式。

### 2. CSS 层叠性与继承性
* **继承性**：子元素会自动继承父元素的某些文本样式属性（如 `color`, `font-size`）。注意，盒子模型的属性（如宽度、边框）通常不会继承。
* **层叠性**：如果给同一个元素设置了相同的属性，后面的样式会覆盖（层叠掉）前面的样式。

### 3. CSS 优先级
当不同的选择器作用在同一个元素上时，听谁的？
`!important` > 行内样式 > ID 选择器 > 类选择器 > 标签选择器 > 继承的样式。

## 任务 3-4 至 3-8 实战：制作“网上花店”专题页

在本项目实战中，我们将利用所学的 CSS 为“网上花店”页面进行装修。

### 页面建设准备工作
首先新建 `flower.html` 和对应的 `style.css`，并在 HTML 中引入 CSS 文件。我们将清除浏览器的默认内外边距，定义基础字体样式。

整个页面分为以下几个模块，我们将一步步进行美化：
1. **标题模块**：设置大标题字号，居中对齐，并调整颜色。
2. **分类模块**：使用并集选择器为花卉分类列表添加统一样式，去除列表默认圆点。
3. **热卖模块**：利用后代选择器，精确控制每张花卉图片下方商品名称和价格的颜色、粗细。
4. **页脚模块**：设置较小的字号和灰色文本，使用 `text-align: center;` 让版权信息居中显示。
