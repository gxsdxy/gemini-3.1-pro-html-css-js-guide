## 任务 1-1 认识网页

当我们每天浏览网页时，你是否好奇过它们是由什么组成的？

### 1. 网页的构成
一个标准的网页通常由以下三部分组成：
* **结构 (Structure)**：如同建筑的骨架，决定了网页中包含哪些内容（文字、图片、视频等）。
* **表现 (Presentation)**：如同建筑的装修，决定了网页的视觉样式（颜色、字体、布局等）。
* **行为 (Behavior)**：如同建筑的电力和供水系统，决定了网页的交互功能（点击按钮发生的反应、动态效果等）。

### 2. 网页的相关名词
在正式开始之前，你需要了解几个常用的专业名词：
* **URL (统一资源定位符)**：网页的地址，比如 `https://www.example.com`。
* **Hyperlink (超链接)**：网页之间的桥梁，点击后可以跳转到其他页面。
* **Server & Client (服务器与客户端)**：服务器用于存储网页文件，客户端（如你的浏览器）用于请求并显示这些文件。

### 3. Web 标准
为了保证网页在不同浏览器中都能正常显示，万维网联盟 (W3C) 制定了一系列标准。遵循 Web 标准可以提高代码的复用性、降低维护成本，并带来更好的 SEO 和访问体验。

## 任务 1-2 网页制作入门技术

前面提到的“结构”、“表现”和“行为”，分别对应了前端开发的三大核心技术。

### 1. HTML5 简介 (结构)
**HTML (超文本标记语言)** 是用来描述网页结构的一种语言。**HTML5** 是它的最新版本，引入了更多语义化标签（如 `<header>`、`<footer>`）和强大的多媒体支持（`<video>`、`<audio>`），让网页结构更清晰、功能更丰富。

### 2. CSS3 简介 (表现)
**CSS (层叠样式表)** 用于控制网页的外观。**CSS3** 带来了圆角、阴影、渐变、动画等强大的视觉效果能力，甚至能取代许多过去需要使用图片或 Flash 才能实现的效果。

### 3. JavaScript 简介 (行为)
**JavaScript (JS)** 是一门编程语言，赋予网页“生命力”。它能让网页对用户的操作做出响应，比如表单验证、数据动态加载、复杂的动画交互等。

### 4. 常用浏览器介绍
浏览器是网页的“翻译官”。作为开发者，你需要了解主流的浏览器及其内核，比如：
* **Google Chrome** (Blink 内核) - *前端开发者的首选，拥有强大的开发者工具。*
* **Mozilla Firefox** (Gecko 内核)
* **Safari** (WebKit 内核) - *Apple 设备的默认浏览器。*
* **Microsoft Edge** (Blink 内核)

> [!TIP] 建议
> 强烈建议在学习和开发过程中使用 **Google Chrome** 或最新的 **Microsoft Edge** 浏览器，它们对最新标准的支持最完善，并且自带极其强大的开发者工具 (F12)。

## 任务 1-3 网页编辑器

工欲善其事，必先利其器。优秀的编辑器能大幅提升你的编码效率。

### 1. 常见网页编辑器介绍
过去大家可能会用记事本或者 Dreamweaver 写代码，但现在的主流是轻量级、插件丰富的现代代码编辑器。

### 2. VS Code 的安装与使用 (强烈推荐 🌟)
**Visual Studio Code (VS Code)** 是微软开源的免费编辑器，目前在全球前端开发者中使用率最高。
* **安装**：访问 [VS Code 官网](https://code.visualstudio.com/) 下载并安装。
* **特点**：速度快、界面美观、插件生态极其丰富。
* **推荐插件**：
  * `Live Server`：一键开启本地服务器并实时预览网页修改。
  * `Chinese (Simplified)`：中文汉化包。
  * `Prettier`：代码自动格式化工具。

### 3. HBuilder X 的安装与使用
HBuilder X 是由 DCloud 推出的一款国产编辑器，对于很多国产框架（如 uni-app）有很好的支持，特点是轻巧、快捷，非常适合国内初学者上手。

## 🤖 AIGC 辅助编程：引入 AI 助手

作为新时代的开发者，学会使用 AI 辅助编程是“降维打击”的技能。

**CodeGeeX / GitHub Copilot** 等 AIGC (人工智能生成内容) 插件可以直接安装在你的 VS Code 中。它们可以：
* **自动补全代码**：根据上下文，预测你接下来要写的代码。
* **代码解释**：选中一段看不懂的代码，让 AI 解释给你听。
* **生成注释**：为你写好的代码自动生成清晰的注释。

> [!IMPORTANT] 作业与练习
> 1. 下载并安装 VS Code 浏览器。
> 2. 在 VS Code 中安装 `Live Server` 和一个你喜欢的 AI 辅助插件（如 CodeGeeX）。
> 3. 尝试在 VS Code 中新建一个 `index.html` 文件，看看 AI 是否能帮你自动补全 HTML 骨架！
