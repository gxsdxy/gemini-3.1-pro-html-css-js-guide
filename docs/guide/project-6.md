# 项目6 “书隅”网站注册页面制作

网页不仅用来展示信息，更需要与用户进行数据交互。无论是登录、注册还是提交问卷，都离不开**表单 (Form)**。在本项目中，我们将学习如何使用表格进行数据排版，以及如何使用表单收集用户输入的内容。

---

## 任务 6-1 认识表格相关标签

在过去，表格常被用来进行网页布局（现已被 CSS 取代）。如今，表格回归了它的本质——**用于展示规整的数据**（如成绩单、财务报表）。

### 1. 表格的基本结构
```html
<table border="1">
    <tr>
        <th>姓名</th>
        <th>年龄</th>
    </tr>
    <tr>
        <td>张三</td>
        <td>18</td>
    </tr>
</table>
```
* `<table>`：定义整个表格。
* `<tr>` (Table Row)：定义表格中的一行。
* `<td>` (Table Data)：定义行中的一个单元格。
* `<th>` (Table Header)：定义表头单元格，文字默认加粗居中。

### 2. 单元格跨行与跨列
* `colspan`：跨列合并。`<td colspan="2">`（横向合并两个单元格）
* `rowspan`：跨行合并。`<td rowspan="2">`（纵向合并两个单元格）

---

## 任务 6-2 CSS 控制表格样式

默认的 HTML 表格比较简陋，我们可以用 CSS 让它变得更专业：
* `border-collapse: collapse;`：这是最常用的表格样式，用于将相邻单元格的双边框合并为单边框。
* 设置 `padding` 增加单元格内文字的呼吸感。
* 使用 `:hover` 伪类实现“鼠标悬停整行变色”的效果。

## 任务 6-3 至 6-5 深入表单 (Form)

表单是收集用户信息的区域。

### 1. 表单的容器：`<form>`
```html
<form action="提交到的服务器地址" method="POST">
    <!-- 表单控件写在这里 -->
</form>
```

### 2. 基础表单控件 (`<input>`)
`<input>` 是最核心的控件，通过改变 `type` 属性，它可以变形为各种输入框：
* `type="text"`：单行文本输入框（如用户名）。
* `type="password"`：密码输入框（输入内容显示为星号）。
* `type="radio"`：单选按钮（注意：同组的 radio 必须有相同的 `name` 属性）。
* `type="checkbox"`：复选框（多选）。
* `type="submit"`：提交按钮。

### 3. 其他重要控件
* `<textarea>`：多行文本输入区域（用于用户留言）。
* `<select>` 和 `<option>`：下拉菜单。

### 4. HTML5 新增表单控件与属性
HTML5 大大增强了表单功能，自带了一些基础校验：
* **新增类型**：`type="email"`, `number`, `url`, `color`, `date` 等（在手机端弹出对应类型的键盘）。
* **新增属性**：
  * `placeholder`：输入框内的提示占位符。
  * `required`：必填项。
  * `disabled`：禁用该输入框。

---

## 任务 6-6 CSS 控制表单样式

原生的表单控件在各个浏览器里长得都不一样，而且通常不太美观。
我们需要使用 CSS 统一其样式：
* 清除默认的边框和轮廓（`outline: none;`）。
* 设置统一的 `height`, `border-radius`, `padding`。
* 使用 `:focus` 伪类，设置输入框获得焦点时（光标在内时）的边框高亮颜色。

## 任务 6-7 至 6-10 实战：“书隅”网站注册页面

结合上述知识，我们将开发“书隅”读书网站的注册页面：
1. **页面布局**：整体使用一个大盒子居中，左右分栏。左侧显示网站愿景图片，右侧为注册表单。
2. **表单开发**：
   * 使用 `<input type="text">` 和 `placeholder` 制作手机号输入框。
   * 使用 `<input type="password">` 制作密码框。
   * 添加一个获取验证码的按钮。
3. **美化**：使用 CSS 控制输入框的间距、聚焦时的阴影效果，并美化底部的“立即注册”按钮，增加悬停反馈。
