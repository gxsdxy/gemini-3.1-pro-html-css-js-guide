## 任务 2-1 认识 HTML

HTML（超文本标记语言）是构建网页的基石，它通过各种**标签 (Tags)** 来描述网页的内容。

### 1. HTML 文档基本格式
一个标准的 HTML5 文档看起来是这样的：
```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>我的第一个网页</title>
</head>
<body>
    内容写在这里...
</body>
</html>
```
* `<!DOCTYPE html>`：声明文档类型为 HTML5。
* `<html>`：根标签，包裹整个网页。
* `<head>`：头部信息（不可见），包含编码、标题等。
* `<body>`：主体内容（可见），你看到的网页都在这里。

### 2. HTML 标签及属性
标签通常成对出现，如 `<p>段落</p>`。部分标签是单标签，如 `<img>`。
标签可以拥有**属性 (Attributes)**，用于提供更多信息：
```html
<a href="https://www.example.com" target="_blank">点击访问</a>
```
* `href` 和 `target` 就是 `<a>` 标签的属性。

### 3. HTML 标签的关系
* **嵌套关系（父子）**：一个标签包含在另一个标签内（例如 `<html>` 包含 `<head>` 和 `<body>`）。
* **并列关系（兄弟）**：同级别的标签（例如 `<head>` 和 `<body>` 就是兄弟关系）。

## 任务 2-2 HTML 文本控制标签

在网页中，文字是最主要的信息载体，HTML 提供了丰富的标签来排版文字。

### 1. 页面格式化标签
* `<h1>` 到 `<h6>`：标题标签，`<h1>` 最大，`<h6>` 最小。
* `<p>`：段落标签，用于包裹一段文字。
* `<br>`：换行标签（单标签）。
* `<hr>`：水平线标签（单标签）。

### 2. 文本格式化标签
* `<strong>` 或 `<b>`：**加粗**。建议使用 `<strong>`，因为它有强调的语义。
* `<em>` 或 `<i>`：*倾斜*。建议使用 `<em>`。
* `<del>` 或 `<s>`：~~删除线~~。
* `<ins>` 或 `<u>`：<u>下划线</u>。

### 3. 特殊字符
有些字符在 HTML 中有特殊含义（比如 `<` 和 `>`），如果你想在网页上显示它们，需要使用字符实体：
* 空格：`&nbsp;`
* 小于号 `<`：`&lt;`
* 大于号 `>`：`&gt;`
* 版权符号 ©：`&copy;`

## 任务 2-3 HTML 图像应用

“一图胜千言”，网页离不开图片。

### 1. 常见图像格式
* `.jpg` / `.jpeg`：适合色彩丰富的照片。
* `.png`：支持透明背景，适合图标和线条图。
* `.gif`：支持简单的动画。
* `.svg`：矢量图，放大不失真。

### 2. 图像标签 `<img>`
```html
<img src="图片路径" alt="图片无法显示时的替代文本" width="300" height="200">
```

### 3. 绝对路径和相对路径
* **绝对路径**：指文件在硬盘上的完整路径或完整的网络地址（如 `https://example.com/logo.png`）。
* **相对路径**：相对于当前 HTML 文件的位置。
  * `./`：当前目录（通常可省略）。
  * `../`：上一级目录。

## 任务 2-4 至 2-7 实战：制作“旅行游记”页面

现在，我们要将上面学到的知识综合起来，制作一个简单的“旅行游记”页面。

### 页面建设准备工作
页面主要分为三个模块：
1. **头部模块 (Header)**：包含游记的网站 Logo 和主标题。
2. **游记模块 (Main Content)**：包含旅行文字介绍和风光图片。
3. **页脚模块 (Footer)**：包含版权信息。

> [!TIP] 动手练习
> 在你的 VS Code 中新建一个 `travel.html`，然后按照上述结构，尝试用 `<h1>`, `<p>`, `<img>` 标签把这三个模块搭建出来！
