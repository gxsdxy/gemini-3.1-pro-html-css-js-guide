# 项目5 “博学谷”网站首页制作

网页中有许多具备明显并列、层级关系的数据，比如导航菜单、新闻列表等。在本项目中，我们将重点学习 HTML 中的列表标签与超链接标签，并结合它们制作出结构规整的“博学谷”网站首页。

---

## 任务 5-1 列表标签

在 HTML 中，列表分为三种主要类型，各有其应用场景。

### 1. 无序列表 (`<ul>` 和 `<li>`)
最常用的列表，列表项前默认显示为小圆点。
```html
<ul>
    <li>苹果</li>
    <li>香蕉</li>
    <li>橘子</li>
</ul>
```
*常用于制作导航菜单、商品分类。*

### 2. 有序列表 (`<ol>` 和 `<li>`)
列表项前带有顺序数字（1, 2, 3...）。
```html
<ol>
    <li>第一步：注册</li>
    <li>第二步：登录</li>
    <li>第三步：学习</li>
</ol>
```
*常用于排行榜、操作步骤说明。*

### 3. 定义列表 (`<dl>`, `<dt>`, `<dd>`)
用于对专业名词或事物进行解释。
* `<dl>`: 定义列表的根标签。
* `<dt>`: 定义项目/名字。
* `<dd>`: 对名字的解释/描述。
*常用于页脚的帮助中心链接、词汇表。*

### 4. 列表的嵌套应用
列表里面还可以包含列表，例如在制作多级导航菜单时：
```html
<ul>
    <li>前端开发
        <ul>
            <li>HTML5</li>
            <li>CSS3</li>
        </ul>
    </li>
</ul>
```

## 任务 5-2 CSS 控制列表样式

列表默认带有圆点/数字以及内置的缩进（内外边距）。我们需要用 CSS 来定制它。

### 1. 去除默认样式
在实际开发中，我们几乎总是需要去掉无序列表前面的小黑点，并清空默认边距：
```css
ul {
    list-style: none; /* 去除列表项目符号 */
    margin: 0;
    padding: 0;
}
```

### 2. 使用背景图片作为列表项目符号
有时候为了美观，我们会用自定义的小图标代替默认圆点：
* 设置 `list-style: none;`
* 给 `<li>` 设置左侧内边距 `padding-left: 20px;`
* 给 `<li>` 设置背景图标 `background: url(icon.png) no-repeat left center;`

## 任务 5-3 超链接标签

互联网的本质就是相互链接的网，超链接 `<a>` 是其中最核心的标签。

### 1. 创建超链接
```html
<a href="URL" target="_blank">链接文字或图片</a>
```
* `target="_blank"`：在新标签页中打开链接。

### 2. 锚点链接
点击后可以快速跳转到当前页面的某个指定位置（例如“回到顶部”或跳转到某个章节）。
1. 在目标位置设置一个 ID：`<h2 id="chapter2">第二章</h2>`
2. 将链接指向该 ID：`<a href="#chapter2">跳转到第二章</a>`

### 3. 链接伪类（控制链接状态）
超链接在不同交互状态下可以有不同的样式，通过 CSS 伪类来实现：
* `a:link`：未访问的链接。
* `a:visited`：已访问的链接。
* `a:hover`：鼠标悬停在链接上时（最常用！）。
* `a:active`：鼠标按下时。
> [!IMPORTANT] 爱恨原则 (LoVe HAte)
> 如果要同时设置这四个状态，必须严格按照 `L-V-H-A` 的顺序书写，否则可能失效。在实际开发中，我们通常主要设置 `a { ... }` 和 `a:hover { ... }`。

## 任务 5-4 至 5-8 实战：“博学谷”首页制作

结合列表和超链接，我们将开发“博学谷”在线学习网站的首页：
1. **导航模块**：使用 `<ul>` 无序列表制作顶部导航条，利用 `float` 变成横向排列，并使用 `a:hover` 增加鼠标悬停时底部出现亮色线条的交互效果。
2. **课程分类模块**：使用多个嵌套或并列的列表构建左侧分类菜单。
3. **热门课程模块**：用列表呈现课程卡片阵列。
4. **版权信息模块**：使用定义列表 `<dl>` 来实现页脚“关于我们”、“帮助中心”等网格布局。
