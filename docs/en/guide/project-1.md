## Task 1-1: Understanding Web Pages

When you browse websites every day, have you ever wondered what they are made of?

### 1. The Composition of a Web Page
A standard web page usually consists of three parts:
* **Structure**: Like the skeleton of a building, it determines what content (text, images, videos) is included in the webpage.
* **Presentation**: Like the interior design, it determines the visual style (colors, fonts, layout) of the webpage.
* **Behavior**: Like the plumbing and electrical systems, it determines the interactive functions (what happens when a button is clicked, dynamic effects, etc.).

### 2. Relevant Terminology
Before we officially begin, you need to understand a few common professional terms:
* **URL (Uniform Resource Locator)**: The address of a web page, such as `https://www.example.com`.
* **Hyperlink**: A bridge between web pages; clicking it allows you to navigate to other pages.
* **Server & Client**: The server stores web files, while the client (like your browser) requests and displays these files.

### 3. Web Standards
To ensure web pages display correctly across different browsers, the World Wide Web Consortium (W3C) has established a series of standards. Adhering to Web Standards improves code reusability, lowers maintenance costs, and brings better SEO and user experiences.

## Task 1-2: Introductory Web Technologies

The "Structure," "Presentation," and "Behavior" mentioned earlier correspond exactly to the three core technologies of frontend development.

### 1. Introduction to HTML5 (Structure)
**HTML (HyperText Markup Language)** is a language used to describe the structure of web pages. **HTML5** is its latest version, introducing more semantic tags (like `<header>`, `<footer>`) and powerful multimedia support (`<video>`, `<audio>`), making the structure clearer and richer in features.

### 2. Introduction to CSS3 (Presentation)
**CSS (Cascading Style Sheets)** is used to control the appearance of web pages. **CSS3** brings powerful visual capabilities such as rounded corners, shadows, gradients, and animations, often replacing effects that previously required images or Flash.

### 3. Introduction to JavaScript (Behavior)
**JavaScript (JS)** is a programming language that brings web pages to "life." It allows web pages to respond to user actions, such as form validation, dynamic data loading, and complex animation interactions.

### 4. Common Web Browsers
Browsers act as "translators" for web pages. As a developer, you need to be familiar with mainstream browsers and their rendering engines, such as:
* **Google Chrome** (Blink Engine) - *The preferred choice for frontend developers, equipped with powerful developer tools.*
* **Mozilla Firefox** (Gecko Engine)
* **Safari** (WebKit Engine) - *The default browser on Apple devices.*
* **Microsoft Edge** (Blink Engine)

> [!TIP] Recommendation
> It is highly recommended to use **Google Chrome** or the latest **Microsoft Edge** for learning and development. They offer the most comprehensive support for modern standards and come with incredibly powerful built-in developer tools (press F12).

## Task 1-3: Code Editors

To do a good job, an artisan needs the best tools. An excellent code editor can significantly improve your coding efficiency.

### 1. Introduction to Common Editors
In the past, people might have used Notepad or Dreamweaver, but the mainstream today leans towards lightweight, highly extensible modern code editors.

### 2. Installing and Using VS Code (Highly Recommended 🌟)
**Visual Studio Code (VS Code)** is a free, open-source editor by Microsoft, and it currently holds the highest usage rate among frontend developers globally.
* **Installation**: Visit the [VS Code Website](https://code.visualstudio.com/) to download and install.
* **Features**: Fast, aesthetically pleasing, and boasts a massive plugin ecosystem.
* **Recommended Plugins**:
  * `Live Server`: Starts a local server with one click for real-time live reloading.
  * `Prettier`: An automatic code formatter.

### 3. Installing and Using HBuilder X
HBuilder X is a domestic editor launched by DCloud (very popular in China). It offers excellent support for many domestic frameworks (like uni-app) and is characterized by being lightweight and fast, making it very suitable for beginners in the region.

## 🤖 AIGC-Assisted Programming: Introducing AI Assistants

As a modern developer, learning to use AI for programming assistance is an invaluable skill.

AIGC (Artificial Intelligence Generated Content) plugins like **CodeGeeX** or **GitHub Copilot** can be installed directly in your VS Code. They can:
* **Autocomplete Code**: Predict what code you want to write next based on context.
* **Explain Code**: Select a confusing block of code and have the AI explain it to you.
* **Generate Comments**: Automatically write clear comments for your code.
