# Frontend Design & Development Curriculum

This repository contains the learning documentation for the **Frontend Design & Development** course, targeted at first-year vocational college students. 
It is based on the textbook *"Web Design & Development Project Course (HTML5+CSS3+JavaScript)"*.

[Read in Chinese (中文)](./README.zh-CN.md)

## Features
- **Project-based Learning**: 8 practical projects covering HTML5, CSS3, and JavaScript.
- **AI-Assisted Learning**: Incorporates AIGC tools like CodeGeeX to boost programming efficiency.
- **Bilingual Support**: English and Chinese content available via VitePress i18n.

## Running Locally

```bash
# Install dependencies
npm install

# Start development server
npm run docs:dev

# Build for production
npm run docs:build
```
